# 🏥 Hospital ER Analysis System

The Hospital ER Analysis System is a web-based application that predicts emergency room waiting time using machine learning. It helps hospitals manage patient flow, monitor doctor availability, and improve resource utilization through data-driven insights.

## 📌Features

- Patient registration and login

- Doctor availability monitoring

- Emergency and standard patient management

- ER wait time prediction using machine learning

- Interactive analytics dashboard with hospital insights

- AI chatbot for user assistance

## ⚙️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python (Flask), Machine Learning (Scikit-learn, Pandas)
- **Database:** Microsoft SQL Server


### Prerequisites

- Python libraries: Flask, Pandas, Scikit-learn, PyODBC
- Modern web browser (Chrome, Edge, or Firefox)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Gayathriv10/Hospital-Emergency-Room-Analysis
cd hospital-emergency-room-analysis
```

2. Install required Python libraries:

```bash
pip install flask pandas scikit-learn pyodbc
```

3. Configure the database connection

Update the SQL Server connection string in app.py.

4. Run the application:

```bash
python app.py
```

5. Open the application in your browser.

```bash
http://127.0.0.1:5000
```
